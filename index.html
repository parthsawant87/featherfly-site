Questions
सवाल

Frequently asked.

अक्सर पूछे जाने वाले सवाल

Do I need a router or internet connection to use the pod?क्या pod use करने के लिए router चाहिए?
No router needed. The WhiteMouse Pod creates its own WiFi hotspot (WhiteMouse-Pod). Your laptop connects directly. AI inference runs locally on the Raspberry Pi — no internet required. Internet is only needed for Claude AI report generation, which falls back to static RCA if offline.
Can I switch between SENTINEL and EduScope on the same pod?क्या एक pod पर दोनों modules use कर सकते हैं?
Yes. Both modules are installed on the pod. Edit one line in startup.sh (MODE="sentinel" or MODE="eduscope") and restart the service. You need an active subscription for each module you use.
What USB microscope works with the WhiteMouse Pod?कौन सा USB microscope काम करता है?
Any USB UVC-compatible digital microscope at 40–400×. We recommend Mustcam or AmScope USB models (₹800–1,400 on Amazon.in). Test it as a webcam on your PC first to confirm compatibility.
How accurate is the AI classification?AI classification कितनी accurate है?
SENTINEL achieves 90–94% accuracy on MobileNetV3 FP32 and 87–93% on YOLOv8s-cls. Training data includes 50% real SEM field-collected images. INT8 quantization drops accuracy by less than 2%. EduScope targets 88%+ across 9 biology specimens.
Is my inspection data sent to the cloud?क्या inspection data cloud पर जाता है?
No. Image capture and AI inference run entirely on the Raspberry Pi inside your facility. Only the text description of a result (not the image) is sent to the Claude API for report generation — and only when you explicitly request a report. Your wafer images never leave the device.
What if the Raspberry Pi Zero 2W is out of stock?अगर RPi Zero 2W stock में नहीं है?
Use Raspberry Pi 3A+ (~₹2,800 on Amazon.in) — same setup, same performance, full-size USB-A ports (no OTG hub needed). The software stack is standard Python — not RPi-specific.
